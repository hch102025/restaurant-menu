# Restaurant Menu

## Build the components

### <b>Main component(App)</b>:

This component imports the menudata and sends it over as a prop to the undercomponents

### <b>Listcomponets(Menu)</b>:

This component recieves the dishes from the meny list and diisplays the list.

### <b>Cardcomponent(Dish)</b>:

This component recieves data for one dish as a prop and shows them in card format. Remember to have title, price, ingredients and category in the display.

## Add styling

1. Use <b>CSS Modules</b> to style application.

2. Create a <b>grid layout</b> for the meny so that the dishes shows in an organized format.

3. Each card should have clear borders, with frames, shadows and easily available font style.

4. Make sure that the title, price, and ingrecients are clearly separated and visually appealing.
